# Godot OpenXR Vendors plugin

The [Godot OpenXR Vendors plugin](https://godotvr.github.io/godot_openxr_vendors/) is a GDExtension
plugin for Godot 4, that adds support for OpenXR vendor extensions, which can't be made part of
Godot itself, because they are too specific to a particular vendor.

The plugin provides the functionality to access vendor-specific OpenXR extensions running on OpenXR
runtimes (e.g: Android XR, Meta, Pico devices).

## Getting the plugin

Official binaries can be found on the Godot [Asset Store](https://store.godotengine.org/asset/godot-xr/godot-openxr-vendors-plugin/)
and [Asset Library](https://godotengine.org/asset-library/asset/5014).

It is also possible to download compiled assets that have been released from [releases](https://github.com/GodotVR/godot_openxr_vendors/releases).

## Documentation

Installation instructions, as well as features and class references can be found in the [documentation](https://godotvr.github.io/godot_openxr_vendors/).
